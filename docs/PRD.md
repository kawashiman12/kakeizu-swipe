# 家系図スワイプ — PRD v2

## 1. プロダクト概要

スマホで直感的に家系図を入力・表示できるWebアプリ。
ローカル保存（localStorage）で動作し、個人情報はクラウド送信しない。

## 2. 基本ルール（家系図のセオリー）

- **スタートは「自分」から**: まず自分→両親→祖父母の順で入力
- **横線＝婚姻、縦線＝子ども**: 図の基本構造
- **兄弟姉妹は左から年齢順**（長男/長女が左）
- **同じ世代は同じ横ラインに揃える**
- **既婚・離婚は記号で表現**（離婚＝「×」）
- **亡くなった人はグレー表示**（性別色は小さい丸印で残す）

## 3. データモデル（v2）

```typescript
interface Person {
  id: string;
  name: string;
  gender: "male" | "female" | "other";
  birthYear?: number;       // 西暦
  birthDate?: string;       // "YYYY-MM-DD" (任意)
  deathYear?: number;       // 没年（西暦）
  deathDate?: string;       // "YYYY-MM-DD" (任意)
  isDeceased: boolean;
  memo?: string;
  birthPlace?: string;      // 出生地（将来用）
}

interface Relationship {
  id: string;
  type: "parent" | "spouse";
  fromId: string;           // parent側 or spouse A
  toId: string;             // child側 or spouse B
  divorced?: boolean;       // 離婚フラグ（spouse時のみ）
}

interface Tree {
  familyName?: string;      // "〇〇家" のタイトル
  persons: Person[];
  relationships: Relationship[];
  updatedAt: string;
}
```

## 4. 画面構成

| パス | 画面 | 説明 |
|------|------|------|
| `/` | ランディング | 新規作成 / 続き / デモ開始 |
| `/editor` | エディター | 人物追加・編集・関係管理 |
| `/tree` | ツリー表示 | フルスクリーン家系図（世代ライン揃え） |
| `/export` | データ管理 | JSON / PDF / 画像 エクスポート |

## 5. 機能要件

### 5.1 人物入力
- **入力項目**:
  - 名前（必須）
  - 性別（必須）: トグル 🔵男性 / 🩷女性 / ⚪その他
  - 生年月日（任意）: **和暦入力 → 西暦自動変換**対応
    - 例: 「昭和33」→ 1958、「令和2」→ 2020
  - 没年月日（任意）: 入力すると自動で故人フラグON
  - メモ（任意）
- **初回起動時は「自分」から入力を促す**ガイド表示
  - Step 1: 自分
  - Step 2: 父・母
  - Step 3: 祖父母
  - 以降自由入力

### 5.2 人物編集
- ツリーまたはリストから人物タップ → 編集フォーム表示
- 全フィールド編集可能
- 「関係を追加」ボタン → 関係追加モーダル
- 既存の関係一覧表示（各関係に「×」削除ボタン）
- 「この人を削除」ボタン → 確認後削除

### 5.3 関係管理
- 親子関係: 方向を明示（「○○が親」「○○が子」）
- 配偶関係: 「💑 配偶者」
- 離婚表示: 配偶関係に「離婚」フラグ → ツリー上で「×」表示

### 5.4 ツリー表示（核心機能）

#### 5.4.1 レイアウト
- **同じ世代を同じ横ラインに揃える**（世代ごとの行）
- 配偶者は横線で接続
- 親子は縦線で接続
- 兄弟姉妹は年齢順（左→右＝年上→年下）

#### 5.4.2 色分け
- 男性: 青系ボーダー＋薄い青背景（`#dbeafe`系）
- 女性: ピンク系ボーダー＋薄いピンク背景（`#fce7f3`系）
- その他: グレー系
- **故人: グレーアウト表示** + 性別色の小丸ドット（🔵/🩷）を角に表示

#### 5.4.3 ボックス内容
- 名前
- 生年（-没年）
- 離婚マーク（×）が必要な場合は接続線上に表示

#### 5.4.4 タイトル
- 「〇〇家 家系図」のヘッダーを表示（familyName設定時）

### 5.5 和暦変換

| 和暦 | 開始年 |
|------|--------|
| 明治 | 1868 |
| 大正 | 1912 |
| 昭和 | 1926 |
| 平成 | 1989 |
| 令和 | 2019 |

- 入力: `昭和33` or `S33` → 表示: `1958年`
- 表示時は西暦。将来的に和暦表示切替も検討

### 5.6 エクスポート
- **JSON**: バックアップ用（既存）
- **PDF**: ツリーを印刷可能な形で出力（将来）
- **JPEG/PNG**: ツリーを画像として保存（将来）

### 5.7 デザインテーマ（将来）
- 和風（明朝体、筆文字風）
- モダン（ゴシック体、フラットUI）
- クラシック（セリフ体、伝統的）
